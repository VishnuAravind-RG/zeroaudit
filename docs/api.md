# Kafka Topics

## Internal Raw Transactions Topic

**Topic Name**

```
zeroaudit.public.transactions
```

This topic is produced by **Debezium CDC**.

Naming pattern:

```
<topic.prefix>.<schema>.<table>
```

---

## Message Format

Produced after the **Debezium ExtractNewRecordState transformation**.

```json
{
  "id": 1,
  "transaction_id": "txn_001",
  "account_id": "acc_12345",
  "amount": 5000.00,
  "balance": 15000.00,
  "timestamp": "2026-03-07T10:00:00Z",
  "bank_signature": "3045022100e476f5059c12b6f5e2f3a5b8b9a0c6d7e8f9a0b1c2d3e4f5a6b7c8d9e0f1a2b3",
  "bank_public_key": "MFYwEAYHKoZIzj0CAQYFK4EEAAoDQgAEKf3Hf2p8yLmzqW5XkYq1nGxP6M9Q8R7S6T5U4V3W2X1Y",
  "metadata": {
    "type": "credit",
    "source": "bank_transfer"
  }

# Public Commitments Topic

**Topic Name**

```

This topic contains **only cryptographic commitments** generated by the Prover.

# Register PostgreSQL Connector

**Endpoint**

```
POST /connectors
```

**Request Body**

Contents of:

```
services/debezium/connector-config.json
```

```json
{
  "name": "zeroaudit-postgres-connector",
  "config": {
    "connector.class": "io.debezium.connector.postgresql.PostgresConnector",
    "database.hostname": "postgres",
    "database.port": "5432",
    "database.user": "audit_user",
    "database.password": "StrongPass123!",
    "database.dbname": "zeroaudit",
    "database.server.name": "zeroaudit",
    "table.include.list": "public.transactions",
    "plugin.name": "pgoutput",
    "slot.name": "zeroaudit_slot",
    "publication.name": "zeroaudit_pub",
    "publication.autocreate.mode": "filtered",
    "topic.prefix": "zeroaudit",
    "transforms": "unwrap",
    "transforms.unwrap.type": "io.debezium.transforms.ExtractNewRecordState",
    "transforms.unwrap.drop.tombstones": "false",
    "value.converter": "org.apache.kafka.connect.json.JsonConverter",
    "value.converter.schemas.enable": "false",
    "key.converter": "org.apache.kafka.connect.json.JsonConverter",
    "key.converter.schemas.enable": "false",
    "snapshot.mode": "initial"
  }
}
```

**Response**

```
201 Created
```

Returns the connector configuration.

# Check Connector Status

**Endpoint**

```
GET /connectors/<connector-name>/status
```

Example:

```
GET /connectors/zeroaudit-postgres-connector/status
```

**Response**

```json
{
  "name": "zeroaudit-postgres-connector",
  "connector": {
    "state": "RUNNING",
    "worker_id": "..."
  },
  "tasks": []
}
```

# Streamlit Auditor Dashboard

The **Auditor Dashboard** runs at:

```
http://localhost:8501
```

The dashboard:

* Consumes **commitments from Kafka**
* Displays **real-time audit verification**
* Does **not expose raw financial data**

No programmatic API is exposed. The interface is UI-only.
